Dominion

Github

Team project over 3 days to recreate the board game Dominion. Rules and a more in-depth description of the game can be found here. A default board of 4 players with basic resource cards, point cards and 6 gameplay cards has been implemented. Current goals are to implement a complete set of rules cards (up to 25 total) and functionality for a variable number of players (2-4). Long term goals include the ability to play across multiple machines using a cloud-stored database.

Twitch Favorites List

Github
CodePen Demo

Exercising with creating DOM elements dynamically with jQuery and using API calls. Uses the Twitch.tv API to generate listings that show the current state of the stream and provides stream information and a live link if the stream is currently on. Currently in a demo state, long term features include implementing a database to store user login information and persist their favorited channels.
